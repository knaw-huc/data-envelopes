# Literature

This page contains links to literature related to data-envelopes and to dataset documentation and contextualisation in general.

Do you know a good reference that isn't listed here? Please [let us know](contact.md).

* [Heger et al, "Understanding Machine Learning Practitioners' Data Documentation Perceptions, Needs, Challenges, and Desiderata", 2022](https://doi.org/10.48550/arXiv.2206.02923)

* [Maemura and Byrne, "Datasheets for Web Archives Toolkit", 2024](https://doi.org/10.22020/rq8z-r112)
* [Pushkarna et al,"Data Cards: Purposeful and Transparent Dataset Documentation for Responsible AI", 2022](https://doi.org/10.48550/arXiv.2204.01075)
